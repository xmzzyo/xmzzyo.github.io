VirtualBox Ubuntu1604 Server 静态ip配置

2018-02-07 Linux 0 Word Count: 345(words) Read Count: 1(minutes)

1 安装ssh

sudo apt-get install openssh-server

2 启动ssh

service sshd start

检查：

ps -e | grep sshd

3 配置桥接网卡

网络连接方式：

  • Bridged networking 桥接模式

  • Network Address Translation (NAT) 网络地址转发

  • Host-only networking 仅主机

  • Internal networking 内部网络

4 配置静态ip

sudo vim /etc/networking/interface

重启网络：

sudo /etc/init.d/networking restart
